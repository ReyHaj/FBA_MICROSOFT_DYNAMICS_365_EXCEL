# FBA_MICROSOFT_DYNAMICS_365_EXCEL

Mini-lab di finanza basato su **Excel** integrato con **Microsoft Dynamics 365**.  
Focus: estrazione dati, pulizia manuale, KPI e dashboard (Pivot/Power Query).

## Contenuti
- Data Quality checklist (Missing, Duplicates, Inconsistent, Invalid)
- Flusso Excel-centrico: Import → Power Query (opzionale) → Tabelle → Pivot Charts
- KPI di esempio: DSO, % On-Time, Aging Buckets, Outstanding
- Report & screenshot per documentazione
